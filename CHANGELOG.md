- Published `saas-audit` as a standalone GitHub repository.
- Added portable Agent Skills metadata in `SKILL.md`.

- Added installation guidance for Claude Code, Codex, Hermes, Anti-Gravity, VS Code/GitHub Copilot and generic Agent Skills clients.
